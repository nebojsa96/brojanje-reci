# brojanje-reci

Konzolna dotNet Core aplikacija sa REST API servisom za prebrojavanje reči u tekstu iz 3 različita izvora (konzola, fajl ili baza).
REST API servis je implementiran kao ASP dotNet Core Web aplikacija.
